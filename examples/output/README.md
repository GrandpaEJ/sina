# Example Outputs

This directory contains the output images generated by running the Sina examples.

## Generated Images

### Basic Shapes

- **Path**: `basic_shapes/basic_shapes.png`
- **Description**: Demonstrates basic shape rendering (rectangles, circles, lines)
- **Run**: `cargo run --bin basic_shapes`

### Vector Paths

- **Path**: `vector_paths/vector_paths.png`
- **Description**: Bezier curves, triangles, stars, and heart shapes
- **Run**: `cargo run --bin vector_paths`

### Complex Shapes

- **Path**: `complex_shapes/complex_shapes.png`
- **Description**: Grid patterns, concentric circles, spirals, checkerboards
- **Run**: `cargo run --bin complex_shapes`

### Benchmark

- **Path**: `benchmark/benchmark_output.png`
- **Description**: Performance benchmark output (1920x1080)
- **Run**: `cargo run --bin benchmark`

## Regenerating Images

To regenerate all example outputs:

```bash
cargo run --bin basic_shapes
cargo run --bin vector_paths
cargo run --bin complex_shapes
cargo run --bin benchmark
```

Or run all at once:

```bash
for example in basic_shapes vector_paths complex_shapes benchmark; do
    cargo run --bin $example
done
```
